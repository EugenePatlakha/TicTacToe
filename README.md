# Tic-Tac-Toe with SignalR in C#

## 📌 Project Description
This is a client-server application for playing Tic-Tac-Toe, developed in C# using **SignalR** for real-time communication.

## 🚀 Technologies
- **C# (.NET)**
- **SignalR**
- **WPF**
